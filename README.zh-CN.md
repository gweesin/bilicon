通过 iconify 在任何地方使用 B 站表情包！

## 动机

B 站有很多有意思的表情包，想在其他应用里使用这些表情包非常困难。这个项目旨在提供一种方法，可以在任何支持 iconify 的应用中使用 B 站的表情包。
